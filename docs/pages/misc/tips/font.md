# font

[JetBrainsMonoNL Nerd Font [ Mono, Propo ]](https://www.nerdfonts.com/font-downloads)

- NL(No Ligatures): 不连字
- Mono(Monospace): 所有字符完全等宽(包括图标)
- NF(Nerd Font): 图标库
- Propo: 字体等宽(图标不等宽)
